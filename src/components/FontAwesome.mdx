<h1>FontAwesome</h1>

## Description

Use this component to quickly add [Font Awesome](https://fontawesome.com/) styles to the `<head/>` element of the page. 

By default, styles are loaded from [bootstrapcdn.com](https://www.bootstrapcdn.com/) CDN.

## Usage

```jsx
import { FontAwesome } from 'react-bootswatch-select';
...
<FontAwesome version={'4.7.0'} />
```

## Props

<Props of={FontAwesome} />

> Note: Expand each prop for more information.

<br/><br/>

The following template is used for the Font Awesome CSS path: `${cdnLocation}/${version}/css/${filename}`
